Definition

Softmodding is where someone changes the code in a device to alter the original intentions of what it was made to do.

Homebrew

One example of softmodding is the Homebrew Channel on the Nintendo Wii. The Homebrew Channel allows users to load Homebrew applications without having to run an exploit every time you want to use it.

You can also Homebrew a Nintendo switch using different possible methods. One of these methods is Softmodding, where you bypass the bootloader on the Nintendo Switch and tell it to boot into a different firmware. Another way is to Hardmod it, which is what people with the V2 Switch have to do.
